# 🧗‍♂️ דני שריון גליצ' - האתר השיווקי הרשמי

ברוכים הבאים למאגר הקוד של האתר השיווקי עבור **דני שריון גליצ'** - מומחי אקסטרים, סנפלינג, פארקי חבלים וסדנאות ODT בשטח.

## 🌟 מאפייני האתר
* **עיצוב מותאם מותג:** פלטת צבעים אורגנית (זהב-חבל, ירוק-יער ושחור-פחם) השאובה ישירות מהלוגו הרשמי.
* **חלוקת קהלי יעד ברורה:** נתיבים שיווקיים נפרדים עבור חברות וארגונים (B2B) ועבור משפחות ופרטיים (B2C).
* **נגישות מלאה:** הטמעה של תוסף נגישות (UserWay) וסרגל נגישות מובנה.
* **הנעה לפעולה דינמית:** טופס יצירת קשר חכם וכפתור וואטסאפ צף לתיאום מהיר.

## 🛠️ טכנולוגיות
* HTML5 / CSS3
* Tailwind CSS (מבוסס CDN)
* FontAwesome (עבור אייקונים)

## 📁 מבנה הפרויקט
* index.html` - קובץ המקור הראשי של האתר.
* `logoHeb.png` - הלוגו הרשמי של החברה (שפת מותג).
